## 🚀 Getting Started

### 1. Clone the Repo / Copy Template
```bash
cd backend
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file based on `.env.sample`:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/your-db
JWT_SECRET=your_secret_key
```

### 4. Start the Server

```bash
npm start
```
